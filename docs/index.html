Supported Project Types

Node.js

Detection: package.json

Installation

bun install (fallback to npm)

Docker

Detection: Dockerfile, docker-compose.yml

Installation

docker-compose up -d

Python

Detection: requirements.txt, setup.py

Installation

Virtual env + pip install

Rust

Detection: Cargo.toml

Installation

cargo build

Go

Detection: go.mod

Installation

go mod tidy
